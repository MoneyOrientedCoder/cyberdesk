# CyberDesk（赛博桌面）

> 无人操作 OS Shell - 自主操作系统

## 项目背景

**核心定位：** 实现无人值守的操作系统自动化管理

**参考竞品：** 

- **AgentGPT** — 浏览器端的 AI Agent
- **OpenManus** — 开源的多代理系统
- **Computer Use (Anthropic)** — 让 AI 直接操作电脑

**目标用户：**
- 开发者/运维工程师
- 需要自动化桌面操作的用户
- 企业流程自动化

**商业模式：**
- 开源版（基础功能）
- Pro 版（高级功能、云服务）
